Watchdog PCB, has 8 outputs and 4 inputs. The first output is used as watchdog output and will be switched off if not triggered regularely. Input 0 is the watchdog readback to check if the watchdog can still switch off the output 0 relay.

The relays are powered via USB 5V, what, depending on the used USB cable, can be a bit less sometimes. If you have problems with that change the USB cable or change the layout:
